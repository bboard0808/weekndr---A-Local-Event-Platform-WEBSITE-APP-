This is my ongoing and ambitious project. I wanted to find a way to showcase my full stack development skills, while also giving back to my local community. 

Here in Charlottesville, it can be hard for one to find events happening locally, which requires various social media to find local events, whether the event be paid, or free. 

I am developing a web app and IOS app for users to locate events happening here in Charlottesville. I want the apps to be user-friendly, as I believe minimal and simple UI is the key to producing a successful application.

This is the website development repository. I am programming the front-end and the back-end. I am utilizing HTML, Javascript, Python inside VS Code.
